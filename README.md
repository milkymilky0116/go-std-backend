# Go Standard Library Backend

# Usage of Gist:

- _-tmp_ boolean
  - Whether running application on Template Mode
- _-dbname_ string
  - Database Name
- _-password_ string
  - Database User Password
- _-port_ int
  - MySQL port (default 3306)
- _-user_ string
  - Database Username (default "root")
